== Description ==

CommentPress for BuddyPress extends the functionality of BuddyPress and BP-Groupblog by enabling groups to have a single CommentPress-enabled blog associated with them.

== Installation ==

1) unzip the .zip file into `/wp-content/plugins/commentpress-bp`
2) Network activate the plugin
3) You are done!
